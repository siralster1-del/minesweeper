# 💣 Minesweeper Game

A fully customizable Minesweeper game built with HTML, CSS, and JavaScript featuring modern UI design and comprehensive game mechanics.

## 🎮 Features

- **Customizable Board Size**: Choose from preset sizes (9x9, 16x16, 20x20) or create custom dimensions
- **Adjustable Mine Count**: Set the number of mines based on your preference and board size
- **Timer Functionality**: Track your game time with a built-in timer
- **Modern UI**: Beautiful gradient design with smooth animations and responsive layout
- **Game Controls**:
  - Left click to reveal cells
  - Right click to flag/unflag cells
  - Hint button for assistance
  - Reset button to start over
- **Win/Lose Detection**: Automatic game state management with victory/defeat screens
- **Responsive Design**: Works on desktop and mobile devices

## 🚀 How to Play

1. Open `index.html` in your web browser
2. Adjust game settings (board size and mine count) if desired
3. Click "New Game" to start
4. Left-click cells to reveal them
5. Right-click cells to flag potential mines
6. Numbers show how many mines are adjacent to that cell
7. Clear all non-mine cells to win!

## 🎯 Game Rules

- Numbers indicate how many mines are in adjacent cells
- Use flags (🚩) to mark cells you think contain mines
- Don't click on mines or you'll lose!
- The timer starts when you make your first move
- Win by revealing all cells that don't contain mines

## 🛠️ Technical Details

- **Frontend**: Pure HTML5, CSS3, and JavaScript (ES6+)
- **No Dependencies**: Runs entirely in the browser
- **Responsive**: Mobile-friendly design
- **Modern CSS**: Uses CSS Grid, Flexbox, and CSS animations
- **Object-Oriented**: Clean JavaScript class-based architecture
